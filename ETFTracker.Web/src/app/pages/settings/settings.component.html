ℹ️ How Exit Tax is calculated

When you record a sell, the system uses FIFO (First In, First Out) to determine which buy lots are being sold. The taxable profit is:

(Sell Price − Weighted Avg Buy Price) × Quantity Sold

The Exit Tax Due is then:

Taxable Profit × Exit Tax %

If no rate is configured for an asset type, the taxable profit is still shown but the Exit Tax Due column will display —.
