## 🔧 Technical Specifications

### Feature: Price Source Tracking
**Purpose**: Track and display which API provided each holding's price

**Supported Sources**:
- Eodhd API (Primary)
- Yahoo Finance API (Fallback 1)
- Database Cache (Fallback 2)
- Unavailable (All fail)

**Data Type**: VARCHAR(50) nullable string
**Storage**: PostgreSQL holdings table
**API Field**: priceSource (string | null)
